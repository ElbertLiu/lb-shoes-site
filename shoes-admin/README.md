# ShoeFactory Admin

管理端后台项目，包含产品管理、分类管理和轮播图管理页面。

## 本地运行

```bash
npm install
npm run dev
```

默认地址：

```text
http://localhost:3001
```

## 环境变量

复制 `.env.example` 为 `.env.local` 后可配置官网地址：

```env
VITE_FRONTEND_URL="http://localhost:3000"
```

## 可用脚本

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
